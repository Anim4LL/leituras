# leituras

Uma biblioteca para livros humanos, reflexivos e sensíveis.

Este repositório foi criado para reunir as obras autorais de Felipe Correia em uma única base visual, com apresentação sóbria, literária e contemporânea.

## Proposta

Mais do que vender livros isolados, a ideia aqui é construir uma biblioteca viva.
Um espaço onde cada obra tenha presença, identidade e lugar dentro de um conjunto maior.

## Estrutura base

- `index.html` — página principal da biblioteca
- `assets/` — capas e imagens usadas no site
- `README.md` — descrição do projeto

## Obras atuais

- **Amar sem possuir**
- **Menos Ruído, Mais Vida**
- **Linguagem Animal**
- **Linguagem Felina**
- **Educar sem apagar**
- **Liderar sem controlar**

## Caminho do projeto

A biblioteca será atualizada progressivamente, com inclusão de novas capas, novas obras, coleções e, em momento posterior, links de compra e páginas específicas.

## Identidade

O projeto busca uma linguagem:

- autoral
- elegante
- humana
- sóbria
- melancólica na medida certa
- visualmente coerente
